FLL_FSHELPER(8)
===============
Kel Modderman <kel@otaku42.de>


NAME
----
fll_fshelper - FULLSTORY live fstab and mountpoint helper


SYNOPSIS
--------
'fll_fshelper' [OPTIONS...]


DESCRIPTION
-----------
fll_fshelper(8) is used for the initial creation of '/etc/fstab' while in the
fll live linux environment. By parsing '/proc/partitions', fll_fshelper can
determine the filesystem type and other attributes via udev\'s utilities and
output a suitable fstab entry for each partition.

fll_fshelper also creates fstab entries for optical and floppy devices.


OPTIONS
-------
-h, --help::
	Print help.

-w, --write-fstab::
	Write to '/etc/fstab' instead of displaying output.

-m, --make-mountpoints::
	Create mount points in /media for partitions written to '/etc/fstab'.

-w, --wanted 'device=1'::
	Do not inhibit partitions on this device from being in the fstab. This
	option may be supplied on the command line multiple times.

-n, --no-swap::
	Do not write a fstab entry for swap partitions.

-u, --uuids::
	Use UUID tokens for mount names when possible.

-l, --labels::
	Use LABELS tokens for mount names when possible.

-f, --fstab '/path/to/fstab'::
	Target fstab file to write to.

-a, --auto::
	Set 'auto' mount option where applicable.

-q, --quiet::
	Surpress output when writing to file.

-d, --debug::
	Enable perl debug output.
