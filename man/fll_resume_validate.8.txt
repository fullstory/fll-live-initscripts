FLL_RESUME_VALIDATE(8)
======================
Kel Modderman <kel@otaku42.de>


NAME
----
fll_resume_validate - validate and set software suspend (resume) configuration


SYNOPSIS
--------
'fll_resume_validate'


DESCRIPTION
-----------
fll_resume_validate(8) validates initramfs-tools(8) resume configuration file,
'/etc/initramfs-tools/conf.d/resume', overwriting it if incorrect based on
information returned from blkid(8).


OPTIONS
-------
'fll_resume_validate' takes no options


SEE ALSO
--------
initramfs-tools(8), blkid(8)
